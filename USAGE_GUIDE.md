# คู่มือการใช้งานโปรแกรมบวกเลข

## ไฟล์ที่สร้างขึ้น

1. **NumberAdder.java** - โปรแกรม Java สำหรับบวกเลข
2. **number_adder.py** - โปรแกรม Python สำหรับบวกเลข

## วิธีใช้งาน

### Java Version
```bash
# คอมไพล์โปรแกรม
javac NumberAdder.java

# รันโปรแกรม
java NumberAdder
```

### Python Version
```bash
# รันโปรแกรมโดยตรง
python3 number_adder.py
```

## ฟีเจอร์ที่มีในโปรแกรม

### 1. บวกเลขสองจำนวน
- รับค่าเลขสองตัวจากผู้ใช้
- แสดงผลลัพธ์การบวก
- รองรับทั้งจำนวนเต็มและทศนิยม

### 2. บวกเลขหลายจำนวน
- ผู้ใช้สามารถกำหนดจำนวนเลขที่ต้องการบวก
- ป้อนเลขทีละตัว
- แสดงผลลัพธ์พร้อมสมการ

### 3. ตัวอย่างการใช้งาน
- แสดงตัวอย่างการบวกเลขแบบต่างๆ
- ไม่ต้องป้อนข้อมูล เหมาะสำหรับดูตัวอย่าง

## ตัวอย่างการใช้งาน

### บวกเลขสองจำนวน
```
เลือกตัวเลือก: 1
ป้อนเลขที่ 1: 15.5
ป้อนเลขที่ 2: 24.3
ผลลัพธ์: 15.5 + 24.3 = 39.8
```

### บวกเลขหลายจำนวน
```
เลือกตัวเลือก: 2
จำนวนเลขที่ต้องการบวก: 4
ป้อนเลขที่ 1: 10
ป้อนเลขที่ 2: 20
ป้อนเลขที่ 3: 30
ป้อนเลขที่ 4: 40
ผลลัพธ์: 10 + 20 + 30 + 40 = 100
```
